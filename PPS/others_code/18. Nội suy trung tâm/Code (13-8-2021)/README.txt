Hướng dẫn sử dụng code:

1. Tải file Central_interpolation.py về máy
2. Tạo file Input.txt 
3. Trong file Input.txt, nhập các dữ liệu sau:
    # Dòng 1: Bảng giá trị của x (Tăng dần và cách đều theo thứ tự từ trái sang phải)
    # Dòng 2: Bảng giá trị của y (Cùng cỡ với bảng dữ liệu của x) 
    # Dòng 3: Giá trị x dùng để nội suy 
    # Dòng 4: Tên công thức ("Stirling" hoặc "Bessel")
4. Chạy chương trình, kết quả chính sẽ được in ở file Output.txt
5. Nhập hàm f(x) sinh ra bảng giữ liệu ở console (nếu có), nếu như không có thì nhập NONE để thoát chương trình
